# VibeWheel

帮助非专业人士使用 AI 编程工具快速开发应用的提示词模板和工作流程。

## 工作流程

### 新手流程（推荐）

1. **下载提示词**：从本项目下载 [轻型技术栈-产品解决方案大师.md](轻型技术栈-产品解决方案大师.md)
2. **生成指令**：
   - 打开 Web 端 AI 工具（推荐 [DeepSeek](https://www.deepseek.com/)）
   - 复制提示词模板到对话中
   - 描述你的需求，AI 会生成面向编程工具的指令文档
3. **生成代码**：
   - 打开 [WaveFox](https://wavefox.ai/)（国内推荐）
   - 将生成的指令文档复制到工具中
   - 工具自动生成项目代码
4. **在线部署**：
   - 在 WaveFox 中直接编辑和调整代码
   - 使用在线部署功能一键部署并获取访问地址

### 进阶流程

1. **下载代码**：从 WaveFox 导出项目代码到本地
2. **本地开发**：
   ```bash
   npm install
   npm run dev
   ```
3. **AI 优化**：
   - 使用本地 AI Coding CLI（Cursor CLI、Windsurf CLI）
   - 或在 IDE（Cursor、VS Code）中使用 AI 辅助编程
   - 导入项目后，AI 会根据代码上下文提供优化建议
